# Tandz-tat (Second Project - First Freelance Project)
![Screenshot of a placeholder page for Tandz-tat.](/Screenshot.png)
## Overview

This is Tandz-Tat, a static TypeScript project designed and made for a friend's tattooing business.

It is a simple site, with only a couple pages meant to display his artwork efficiently to people who would want one

It was important during the development of the site to not be too flashy with design, and let his artwork speak for itself.

From this project, the most technical side of it is the Instagram API, which calls all his post information, and displays them in the gallery page. Since it uses his personal Meta API key, it is hidden from the public repository.
